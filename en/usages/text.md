## Best Text Control Features

- `Alt` + `ikjl` - power arrow
- `Alt` + `esdf` - normal arrow
- `Alt` + `bi` - backspace, delete
- `Alt` + `a` - select line
- `Alt` + `w` - select word


<br/>

- `ctrl` means `power`
- `shift` means `speed` with magic key
- `shift` means `speed` with `Magic key` and Original keybinding with Window Key
