# 0.8.0

## New feature

### 향상된 텍스트 기능

- `Alt` + `h;`
  - `h;` - 문서단위 이동
- `Alt` + `'Enter`
  - `'Enter` - 관통엔터 & 관통백엔터

### `ijklh;`

- shift를 같이 누르면 원래 단축키가 실행됩니다
- ctrl를 같이 누르면 바로 가속된 상태로 이동합니다

### `uom`

- shift를 같이 누르면 shift click이 가능합니다 (파일 멀티 선택 등)
- ctrl를 같이 누르면 ctrl를 click이 가능합니다 (파일 개별 선택 등)
